🟦 Friendly Ground Marker

        
a-f-G

        
Manually placed friendly ground position marker with archive, link, and creator. Typical ATAK long-press drop-pin.

      

      

        
📍 SA Beacon (Infantry)

        
a-f-G-U-C

        
Situational awareness beacon from a human ATAK user. The most common CoT message type on a TAK network.

      

      

        
🤖 UGV Position Report

        
a-f-G-U-C-R

        
Autonomous ground robot classified as Combat Reconnaissance. Verified from ATAK stream.

      

      

        
🚁 UAV — Rotary Wing (Anafi)

        
a-f-G-U-C-V-U-R

        
Quadcopter/rotary UAV as ground unit organic asset. Verified from ATAK.

      

      

        
🚁 UAV — Generic

        
a-f-G-U-C-V-U

        
Generic UAV, wing type unspecified. Use when platform type is unknown.

      

      

        
✈️ UAV — Fixed Wing

        
a-f-G-U-C-V-U-F

        
Fixed wing UAV as ground unit organic asset. Verified from ATAK.

      

      

        
🚁 UAV — Rotary RPV (Air)

        
a-f-A-M-H-Q

        
Rotary drone under Air dimension. Use when operating around manned aviation / ATC.

      

      

        
🟥 Hostile Vehicle

        
a-h-G-E-V

        
Hostile ground vehicle marker manually placed on map by an operator.

      

      

        
💬 GeoChat Direct Message

        
b-t-f

        
TAK direct message from operator to robot. Message UUID is regenerated each send.

      

      

        
❓ Unknown Ground Contact

        
a-u-G

        
Unknown ground contact — unidentified entity, pending classification.

      

      

        
⬜ Spot Map Marker (White)

        
b-m-p-s-m

        
Persistent white dot dropped on the map. 1-year stale, archived.

      

      

        
🚩 Mission Waypoint (GOTO)

        
b-m-p-w-GOTO

        
Green flag waypoint. 5-minute stale. Callsign auto-generated from creator + timestamp.

      

      

        
📏 Phase Line

        
u-d-f

        
MIL-STD-2525C phase line (G*G*GLP). "PL" label at each endpoint.

      

      

        
🔷 NAI Polygon

        
u-d-f

        
Named Area of Interest (G*G*SAN). Closed polygon with transparent fill.

      

      

        
🟠 Drawing Circle

        
u-d-c-c

        
Orange circle with semi-transparent fill. Radius in <shape><ellipse>.

Display label shown on the map pin in ATAK.
Stable UID for this marker. Reusing the same UID updates the existing pin instead of creating a new one.
Signed 32-bit ARGB integer. -1 = 0xFFFFFFFF = full white.

📋 Preset CoT Examples — Send Raw XML Directly
These presets bypass the form fields — timestamps are set to now at send time. Click a preset to preview it, then send it directly or copy it.
🟦 Friendly Ground Marker
a-f-G
Manually placed friendly ground position marker with archive, link, and creator. Typical ATAK long-press drop-pin.
📍 SA Beacon (Infantry)
a-f-G-U-C
Situational awareness beacon from a human ATAK user. The most common CoT message type on a TAK network.
🤖 UGV Position Report
a-f-G-U-C-R
Autonomous ground robot classified as Combat Reconnaissance — represents role/function (mobile camera recon) not platform type. Verified from ATAK.
🚁 UAV — Rotary Wing (Anafi)
a-f-G-U-C-V-U-R
Quadcopter/rotary UAV as ground unit organic asset. Correct classification for Parrot Anafi. Verified from ATAK.
🚁 UAV — Generic
a-f-G-U-C-V-U
Generic unmanned aerial vehicle, wing type unspecified. Use when platform type is unknown. Verified from ATAK.
✈️ UAV — Fixed Wing
a-f-G-U-C-V-U-F
Fixed wing UAV as ground unit organic asset. Verified from ATAK.
🚁 UAV — Rotary RPV (Air)
a-f-A-M-H-Q
Rotary drone under Air dimension. Use when operating around manned aviation / ATC environments. Verified from ATAK.
🟥 Hostile Vehicle
a-h-G-E-V
Hostile ground vehicle marker manually placed on map by an operator.
💬 GeoChat Direct Message
b-t-f
TAK direct message from operator to robot. Edit the remarks text to change the message content. Message UUID is regenerated each send.
❓ Unknown Ground Contact
a-u-G
Unknown ground contact — unidentified entity, pending classification.
⬜ Spot Map Marker (White)
b-m-p-s-m
Persistent white dot dropped on the map. 1-year stale, archived. Reuse the same UID to move an existing pin.
🚩 Mission Waypoint (GOTO)
b-m-p-w-GOTO
Green flag waypoint from the Mission symbol set. 5-minute stale. Callsign auto-generated from creator callsign + timestamp.
📏 Phase Line
u-d-f
MIL-STD-2525C phase line (G*G*GLP). Shows as a labeled line with "PL" at each end. Add more <link point> elements for a polyline.
🔷 NAI Polygon
u-d-f
Named Area of Interest (G*G*SAN). Closed polygon with transparent fill. First vertex repeated at end to close shape.
🟠 Drawing Circle
u-d-c-c
Orange circle with semi-transparent fill. Radius in <shape><ellipse>. Includes embedded KML style block for export compatibility.
